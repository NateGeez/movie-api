This application provides the user with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description endpoint request method
Return all movies /movies GET
Return data about a single movie /movies/title GET
Return data about a genre /movies/title/genre GET
Return data about a director /movies/title/director GET
Allow new users to register /users POST
Allow users to update info /users PUT
Allow users to add a movie to their list of favorites /movies POST
Allow users to remove a movie from their list of favorites /movies DELETE
Allow existing users to deregister /users DELETE
